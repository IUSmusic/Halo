Pinned surface keyboard

GhostBoard Surface Pad

First the user draws a rectangle in the air over any visible surface. The app turns that shape into a neon 2D keyboard pad, then pins it in place for typing while the camera remains still.

External cameras work too as long as the browser exposes them as video devices. After the first camera permission grant, use the camera selector to switch sources.

How it works

  1. Start the camera and choose the camera source you want.
  2. Press Start drawing rectangle and trace a wide rectangle in the air over the surface where the keyboard should live.
  3. Press Finish rectangle to preview the detected keyboard pad.
  4. Press Pin keyboard to lock it in place.
  5. Type by moving your fingertip over keys and using pinch, dwell, or both.

Best accuracy comes from a stationary camera, a clearly visible surface, and a rectangle drawn as neatly as possible.

Product direction

This version treats the pinned keyboard as a 2D pad in the camera image, which is lightweight and deployable on GitHub Pages. A later version can add true surface tracking, corner refinement, keystone correction, and stronger tap recognition.
