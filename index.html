How to use

  1. Start the camera.
  2. Use One-hand sculpt to pinch, move, resize, and rotate the keyboard with one hand, or use Rectangle trace to draw a placement area.
  3. Pin the keyboard.
  4. Type with any fingertips. Press mode uses a short poke toward the camera plane; dwell and pinch are optional.
  5. Optional voice commands: say type b, press enter, bravo, space.
